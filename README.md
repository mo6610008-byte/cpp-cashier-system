# cpp-cashier-system
# 🎰 C++ Cashier & Receipt System

A simple, robust console-based Cashier/POS application built with C++.

## ⚡ Features
- **OOP Structure:** Clean object design for products (RAM, GPU, Keyboard).
- **Real-Time Stamps:** Integrated `<ctime>` for live date & time generation on receipts.
- **Dynamic Tax Calculation:** Calculates different tax rates per product automatically.
- **Fail-Safe Input Validation:** Handles invalid menu choices and character inputs using `cin.fail()`.

## 🛠️ Built With
- C++
- Standard Template Library (STL)
